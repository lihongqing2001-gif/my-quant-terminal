本终端的核心基于经典的 均值回归策略 (Mean Reversion)，并结合 布林带 (Bollinger Bands) 进行了针对性优化。1. 策略构成 (Composition)该策略由三条轨道组成的“价格通道”构建而成：🟦 价值中枢 (Middle Band)：定义： 过去 $N$ 天的收盘价移动平均线 (MA)。作用： 代表市场的“公允价值”或“引力中心”。🟥 压力上轨 (Upper Band)：定义： 中轨 + $K$ 倍的标准差 ($\sigma$)。作用： 统计学上的“超买区”，价格触及此处通常意味着短期过热。🟩 支撑下轨 (Lower Band)：定义： 中轨 - $K$ 倍的标准差 ($\sigma$)。作用： 统计学上的“超卖区”，价格触及此处通常意味着短期恐慌过度。2. 运行原理 (Principle)基于 正态分布 (Normal Distribution) 的统计学原理：回归属性： 大宗商品价格虽然短期会剧烈波动，但长期倾向于围绕“价值中枢”波动。像一根橡皮筋，拉得越长，回弹的力度越大。小概率事件： 当价格突破上轨或跌破下轨时，属于统计学上的“异常事件”（Outlier）。交易逻辑：做空信号： 当价格冲破上轨（橡皮筋拉伸到极限），预期价格会回落，执行卖出。做多信号： 当价格跌穿下轨（橡皮筋压缩到极限），预期价格会反弹，执行买入。止盈逻辑： 当价格回归中轨（回到舒适区），橡皮筋张力消失，平仓离场。📖 用户使用手册 (User Manual)第一阶段：策略实验室 (Research Lab)用于寻找当前最适合该品种的参数。选择品种：在左侧侧边栏选择您关注的主力品种（如：螺纹钢、玻璃、棕榈油）。注：系统会自动加载该品种在（上期所/郑商所/大商所）的真实手续费标准。启动极速扫描：点击主界面的 “🚀 启动极速扫描” 按钮。系统将在 1 秒内回测 MA10 到 MA90 周期与 1.8 到 2.8 阈值的所有组合。选择模型：扫描完成后，系统会推荐三张“王者卡片”：🔥 进取型： 赚钱最多，适合激进风格。🛡️ 防御型： 夏普比率最高（回撤小），适合稳健风格。⚖️ 平衡型： 综合评分最高，推荐首选。一键部署：点击卡片下方的 “👉 部署” 按钮，直接携带该参数跳转至实盘界面。第二阶段：实盘指挥部 (Live Cockpit)用于盘中监控与执行。信号看板 (Dashboard)：⚪ 灰色 (观望)： 价格在通道内，无需操作，耐心等待。🔴 红色 (做空)： 价格触碰上轨，出现做空信号。🟢 绿色 (做多)： 价格触碰下轨，出现做多信号。交易计划区：动作 (Action)： 明确告诉你是 Buy (买) 还是 Sell (卖)。挂单 (Limit)： 建议的挂单价格（精确到个位）。仓位 (Position)： 基于内置风控模型（默认 50万本金，20% 单笔风险敞口）计算出的建议手数。防止重仓暴雷。止盈 (TP)： 这一单的目标价格（中轨）。图表确认：观察下方图表，橙色点为最新价。确认其是否真的处于通道边缘，辅助人工二次确认。⚠️ 风险提示 (Disclaimer)历史不代表未来： 回测数据基于过去 5 年的历史行情，市场结构变化可能导致策略失效。单边行情风险： 均值回归策略最怕“极端的单边暴涨暴跌”（趋势行情）。如果遇到这种行情，价格可能会沿着上轨一直涨（钝化）。建议： 严格设置止损（例如亏损超过 5% 无条件离场）。网络延迟： 网页版数据受限于网络环境，可能存在秒级延迟，高频交易请谨慎使用。🔧 部署指南 (For Developers)本应用基于 Python Streamlit 框架构建。依赖库： streamlit, akshare, pandas, numpy, matplotlib, scipy启动命令： streamlit run app.py数据源： 新浪财经期货接口 (Sina Futures API)
